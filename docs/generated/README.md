# Generated Gnosis Math Docs

Parent: [Gnosis Math Docs](../README.md)

Generated manifests for wiki and documentation pipelines.

- [`wiki-topic-manifest.json`](./wiki-topic-manifest.json) - scored topic
  manifest produced from `docs/WIKI_TOPICS.md` and local Lean anchors.
- [`wiki-generation-list.md`](./wiki-generation-list.md) - concrete generation
  queue including atlas topics, topology-derived pages, and spiritual/witness
  pages.
- [`wiki-topic-rejections.json`](./wiki-topic-rejections.json) - dump-derived
  candidates rejected because no local anchor matched.
