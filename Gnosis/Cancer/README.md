# Gnosis Cancer Modules

Parent: [Gnosis theorem tree](../README.md)

This directory contains Init-facing cancer topology and Sandy/Fork Cancer proof scaffolds.
The modules focus on bounded formal claims: topological deficit accounting, treatment
ordering, prediction witnesses, confinement, and cancer-as-compromised-cellular-mesh
security hypotheses.

## Modules

- `CancerConfinement.lean` -- confinement ledger anchor.
- `CancerHackSearch.lean` -- topology-to-intervention search router: classify compromised topology, select the matched treatment mechanism, and reject mismatches by antitheorem.
- `CancerPredictions.lean` -- first prediction ledger anchor.
- `CancerPredictions2.lean` -- second prediction ledger anchor.
- `CancerPredictions3.lean` -- epigenetic drift, dormancy, forced vent activation, Warburg overhead, and abscopal propagation.
- `CancerPredictions4.lean` -- fourth prediction ledger anchor.
- `CancerTopology.lean` -- vent-capacity, deficit, mutation, and restoration ordering witnesses.
- `CancerTreatments.lean` -- treatment ordering primitives used by topology witnesses.
- `CancerMeshSecurity.lean` -- cancer-as-compromised-cellular-mesh security hypotheses. It now carries named Init-only witnesses for every Sandy opportunity row: immune sieve liveness, boundary mismatch, kill-switch persistence, sentinel memory, checkpoint bypass, Warburg overhead, abscopal transfer, route poison, Ki timing, tissue topology, cleanup backlog, quarantine, honeytokens, mutation kill score, anti-theorem triage, zero-trust witnesses, immune memory, patch folding, metastasis cutsets, angiogenesis, exosomes, stem persistence, heterogeneity, exhaustion, leaky senescence, repair gaps, epigenetic drift, synthetic lethality, autoimmunity risk, therapy resistance, collective shielding, cyclic windows, and living-off-the-land repair hijack.
- `CancerTreatmentHypotheses.lean` -- treatment-hypothesis brainstorm generated from the mesh-security formalization. Each candidate has pass conditions and antitheorem failure gates so unsupported mechanisms fail before promotion.
- `CancerTherapyMatrix.lean` -- all-topology / all-therapy matrix layer. It scores therapy/topology cells, admits matched cells, and gates the nuclear combination by modeled benefit over risk.
- `OffShelfDrugScreen.lean` -- finite off-the-shelf / off-patent repurposing screen. It enumerates 24 common drug classes, maps each to a cancer-information topology, and proves pursue/reject gates for evidence and risk.
