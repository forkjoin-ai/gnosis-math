# Second Timothy Witnesses

[Parent](../README.md)

Lean witnesses extracted from Second Timothy. New Second Timothy witnesses should be added in this directory and imported as `Gnosis.Witnesses.Bible.SecondTimothy.<Module>`.

## Witnesses

- `SecondTimothySourceQualityWitness`: book-level guarded-deposit invariant and abandonment/drift counterproof.
- `SecondTimothySoundMindDepositWitness`: 2 Timothy 1.
- `SecondTimothyUnboundWordWorkmanWitness`: 2 Timothy 2.
- `SecondTimothyPerilScriptureWitness`: 2 Timothy 3.
- `SecondTimothyFinalChargeCrownWitness`: 2 Timothy 4.
