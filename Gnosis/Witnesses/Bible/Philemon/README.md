# Philemon Witnesses

[Parent](../README.md)

Lean witnesses extracted from Philemon. New Philemon witnesses should be added in this directory and imported as `Gnosis.Witnesses.Bible.Philemon.<Module>`.

## Witnesses

- `PhilemonSourceQualityWitness`: book-level voluntary reconciliation and debt-transfer spine.
- `PhilemonLoveRefreshWitness`: Philemon 1:1-7.
- `PhilemonOnesimusDebtWitness`: Philemon 1:8-25.
