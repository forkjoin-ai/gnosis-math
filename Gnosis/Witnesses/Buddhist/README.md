# Buddhist Witnesses

[Parent](../README.md)

Lean witnesses extracted from Buddhist source documents. New Buddhist witnesses should be added in this directory and imported as `Gnosis.Witnesses.Buddhist.<Module>`.

## Witnesses

- `DhammacakkappavattanaWheelWitness`: `docs/ebooks/source-texts/dhammacakkappavattana-sutta-thanissaro.txt`, lines 1-16, the first-turning wheel as exhaustion protocol: two extremes are rejected, the eightfold middle path opens vision and knowledge, the four truths close only after three rounds each, and awakening is not claimed until the 4x3 wheel is pure.
- `DhammacakkappavattanaPaliEnglishWitness`: `docs/ebooks/source-texts/dhammacakkappavattana-sutta-pali-english.pdf`, `pdftotext` lines 25-119, 122-235, and 239-333, cross-translation closure: introduction, English rendering, and Pali formula all converge on the same `tiparivaṭṭaṃ dvādasākāraṃ` wheel, with the deva echo ladder as distributed propagation.
