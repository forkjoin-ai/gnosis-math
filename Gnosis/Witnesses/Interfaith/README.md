# Interfaith Witnesses

[Parent](../README.md)

Lean witnesses extracted from interfaith and comparative source documents. New interfaith witnesses should be added in this directory and imported as `Gnosis.Witnesses.Interfaith.<Module>`.

## Witnesses

- `ScripturesOfWorldCatalogOpeningWitness`: `docs/ebooks/source-texts/interfaith-scriptures-of-the-world.pdf`, pages 1-2, a bibliographic holdings ledger rather than a scripture body: title/author/location/publisher/ISBN/edition columns, Great Room convergence, translator/editor role metadata, missing identifier tolerance, multi-line extraction gaps, and alphabetic arrangement across traditions.
